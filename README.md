# searchNavigation
## Hunter-InfoGather-Helper

一个能根据自定义模板自动生成相应搜索语法链接,然后在某个栏目一键批量打开多个网页的工具

--------------------搜索引擎---------------------------------
- [ ] Google
```
查找二级域名
site:target.com -inurl:www
查找后台
site:target.com intitle:(manager | admin | 管理登录 | login) | inurl:(admin | manage | login)
查找目录索引
site:target.com intitle:index of /
```
